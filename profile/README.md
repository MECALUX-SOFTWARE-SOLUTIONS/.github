# Mecalux Software Solutions (MSS)

En **Mecalux Software Solutions** diseñamos, desarrollamos y operamos **software de supply chain** como una **plataforma tecnológica**, con foco en ingeniería, fiabilidad y seguridad. Nuestro trabajo combina desarrollo de producto, operación 24x7 y mejora continua para entornos críticos.

> 📍 **Sedes**: Gijón · Badajoz · Salamanca
> 🏢 **Modalidad**: Híbrido  
> 👥 **Equipo**: Más de 330 profesionales

---

## Qué hacemos

Construimos soluciones que impactan directamente en la eficiencia operativa de las cadenas de suministro:

- **Ingeniería de software**: desarrollo de producto y evolución funcional
- **SRE / Plataforma**: operación cloud, automatización, observabilidad y resiliencia
- **Seguridad**: prácticas de ciberseguridad aplicadas a plataforma y ciclo de vida
- **Datos e IA**: analítica aplicada, modelos y capacidades de automatización
- **Rendimiento**: optimización y monitorización continua (DBA / performance)
- **Operaciones e implantación**: despliegues, soporte **24x7** y estabilización en producción

---

## Cómo trabajamos

Priorizamos prácticas de ingeniería y operación orientadas a calidad y sostenibilidad:

- **Infraestructura como código (IaC)** y estandarización de entornos
- **CI/CD**, revisiones de código y criterios de calidad
- **Observabilidad** (métricas, logs y trazas) como base para operar
- **Gestión de fiabilidad**: SLOs/SLAs, respuesta a incidentes y mejora post-mortem (blameless)
- **Seguridad por diseño**: controles prácticos, gestión de secretos y hardening

## Contribución

En repositorios públicos, aceptamos contribuciones mediante Pull Request:

1. Fork del repositorio
2. Crea una rama con un nombre descriptivo (`feature/...` o `fix/...`)
3. Abre un Pull Request con contexto, alcance y evidencia de pruebas

Consulta `CONTRIBUTING.md` en cada repositorio para detalles.
